# Vector Calculator
A Python tool that performs 2D vector calculations using both manual 
implementation and NumPy, with Matplotlib visualization.

## Features
- Vector formation from two points
- Norm (magnitude) calculation — manual and via NumPy
- Dot product
- Cross product
- 2D vector visualization with Matplotlib

## Implementation
Each operation is implemented twice: manually using Python and the math 
library, and via NumPy, allowing direct comparison between the two approaches.

## Technologies
- Python
- NumPy
- Matplotlib

## How to run
python vector_calculator.py

## Example input
Write the values of x and y of P1(x,y), separated by space: 1 2
Write the values of x and y of P2(x,y), separated by space: 4 6
